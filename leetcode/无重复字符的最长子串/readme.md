# diff算法相关的LeetCode

- diff算法如何实现的
  - 组件要渲染的时候，先生成虚拟DOM(VDOM) 树 -> 显示到界面上 -> 数据更新 -> 生成新的VDOM树 -> diff算法比较新旧VDOM树，收集它们差异的过程就是diff的过程也就是patch -> 生成patch -> 更新到界面上

- ABCD ACBD 同层比较
- 简单diff算法
  - 1. 从头开始比较，A === A，继续比较
  - 2. B !== C，将B标记为删除，C标记为新增
  - 3. C === C，继续比较
  - 4. D === D，继续比较
  - 5. 比较结束，将B删除，C新增
